Saat user memilih output:
- Broadcast Telegram/WA
- Friendly WhatsApp
- Announcement Style
- atau format yang ditujukan untuk WhatsApp

MAKA format markdown WA wajib menggunakan SINTAKS WHATSAPP ASLI, BUKAN markdown web/html.

JANGAN gunakan:
# Heading
## Heading
### Heading
- bullet markdown web
render heading besar HTML
atau styling rich text web lainnya.

Karena output akan dipakai langsung di WhatsApp.

Gunakan format WhatsApp native berikut:

FORMAT BOLD:
*teks bold*

FORMAT ITALIC:
_teks italic_

FORMAT STRIKE:
~teks~

FORMAT MONOSPACE:
```teks```

Bullet gunakan:
• atau -
bukan markdown nested aneh.

Separator gunakan:
──────────
atau
———

JANGAN gunakan:
# ##
render heading besar
HTML heading
auto typography web

Karena akan rusak saat dicopy ke WhatsApp.

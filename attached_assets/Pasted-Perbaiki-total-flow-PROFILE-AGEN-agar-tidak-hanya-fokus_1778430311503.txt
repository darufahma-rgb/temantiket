Perbaiki total flow PROFILE AGEN agar tidak hanya fokus ke penjualan produk, tetapi juga otomatis menghitung dan menampilkan penghasilan dari tugas lapangan (VOA / operasional / handling bandara).

MASALAH SAAT INI:
- profile agent hanya membaca komisi sales/order
- fee agent lapangan tidak masuk
- VOA operational fee tidak sinkron
- total komisi jadi salah
- dashboard agent tidak merepresentasikan performa nyata agent lapangan
